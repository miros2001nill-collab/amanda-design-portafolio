# Miroslava - Portafolio de Diseño Gráfico

> Portafolio profesional para una diseñadora gráfica. Diseño moderno, responsivo y con modo oscuro.

## Descripción

Este es un portafolio profesional para una diseñadora gráfica que muestra su trabajo, servicios, blog y contacto. El sitio está diseñado con un enfoque moderno, minimalista y completamente responsivo.

## Características

- ✅ Diseño responsivo (mobile-first)
- ✅ Modo oscuro/claro
- ✅ Menú de navegación con submenú desplegable
- ✅ Menú hamburguesa para dispositivos móviles
- ✅ Galería de proyectos con imágenes
- ✅ Carrusel de videos
- ✅ Blog con artículos
- ✅ Formulario de contacto
- ✅ Optimizado para SEO

## Tecnologías Utilizadas

| Tecnología       | Uso                  |
|------------------|----------------------|
| **HTML5**        | Estructura del sitio |
| **CSS3**         | Estilos y diseño     |
| **JavaScript**   | Interactividad       |
| **Font Awesome** | Iconos               |
| **Google Fonts** | Tipografías          |

### Tipografías

- **Space Grotesk** → Títulos y menú
- **Inter** → Textos normales
- **Manrope** → Subtítulos y badges

| Color           | Hex       | Uso                |
